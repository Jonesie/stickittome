# Sample poses

Generated by `scripts/make_sample_poses.py`. Each sample has both a rendered
`.png` and a re-editable `.pose.json`.

## Files

| File                   | Size      | What it is                                              |
| ---------------------- | --------- | ------------------------------------------------------- |
| `standing.png`         | 512 × 512 | Neutral standing pose, centred                          |
| `walking.png`          | 512 × 768 | Simple walking pose — opposite arm and leg forward      |
| `arms_up.png`          | 512 × 512 | Arms raised (Y-shape) pose                              |

## Regenerating

```bash
python scripts/make_sample_poses.py
```

## Using as ControlNet input

1. Pick one of the `.png` files.
2. In ComfyUI, attach a ControlNet loader + apply node pointing at the file.
3. Set `control_net` to `control_v11p_sd15_openpose` (or your preferred pose
   model) and set the strength to taste.

The matching `.pose.json` lets you open the same pose in the desktop app,
tweak it, and re-export.
